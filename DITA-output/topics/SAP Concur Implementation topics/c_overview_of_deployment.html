Example: Two-Level list:

Your company may have different departments in each company region. In order to make sure the correct departments show for the regions, you can configure a multiple level list using the Region and Department fields.

The Region field would be the first or top-level list field. When you select Department as the second level field, you're specifying that each list item in the Region field will contain a list of valid departments.
